# Tag 标签

进行标记和分类的小标签，用于标记事物的属性和维度，以及进行分类。

## 规则

标签文字必须显示完全。

## props

| 属性 | 说明 | 类型 | 默认值 | 备注 |

| ---- | ---- | --- | ------- | --- |

| small | 小号标签 | Boolean | false | - |

| disabled | 是否不可用 | Boolean | false | - |

| closable | 是否关闭（非 disabled small 状态） | Boolean | false | - |

| selected | 是否默认选中 | Boolean | false | - |
